# Flexbox 内容换行

使用 flexbox 布局的时候，默认情况下，在容器宽度不够时，可能就会出现这样的情况

这个主要是因为 flex-wrap 的默认值是 nowrap 需要这样改变一下值

.options-list {
  display: flex;
  flex-wrap: wrap;
}

# 长文本内容

文本过长 换行或者截断并已 ... 的形式出现 

.username {
  /* 不换行 */
  white-space: nowrap;
  /* 溢出隐藏 */
  overflow: hidden;
  /* 省略号显示 */
  text-overflow: ellipsis;
}

# 图片处理

防止图像被拉伸或压缩 这种情况一般是在是图片由用户上传，或者图片未按规定比例使用导致的。

.card__thumb {
  object-fit: cover;
}

# CSS 变量安全值

当 CSS 变量值无效时而使用一个“安全值”来保证某个属性值

/* 如果第一个参数引用的自定义属性无效，则该函数将使用第二个值 */
.message__bubble {
  max-width: calc(100% - var(--actions-width, 70px));
}

# 毛玻璃

/* 径向渐变 */
background-image: radial-gradient(transparent 1px, #fff 1px);
/* 背景缩放 */
background-size: 4px 4px;
/* 元素后面区域添加 饱和度 和 模糊效果 */
backdrop-filter: saturate(50%) blur(4px);
